# hololis-craft MythicMobs Packs

This repository contains a collection of MythicMobs packs, configuration, and resource files used for creating custom mobs, items, skills, and loot tables for Minecraft servers.

## Prerequisites

- Bun

## Quick setup

1. Clone this repository:

```bash
git clone https://github.com/hololis-craft/hololis-craft-MythicMobs

cd hololis-craft-MythicMobs
```

2. Install dependencies:

Using Bun:

```bash
bun install
```
